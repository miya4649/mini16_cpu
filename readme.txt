Mini16-CPU

Very small and high performance CPU

Features:
16bit RISC like ISA
Variable register, data width (16, 32, 64bits ...)
7 stage pipeline
Highly parameterized design

FPGA logic usage:
DE0-CV: 105 ALM (280 LE) (CPU only)
Kintex UltraScale+ (xcku3p-ffva676-3): 45 CLB (192 LUTs)

Maximum frequency:
DE0-CV: 240 MHz (Proofed)
Kintex UltraScale+: 710 MHz (Simulation)
